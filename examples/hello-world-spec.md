# Hello World Task Spec

Write a Python script that:
1. Accepts a single command-line argument: a string to process
2. Prints the string reversed, uppercased, and its character count
3. Saves the result to `output.txt` in the current directory

Example:
```
$ python3 hello.py "hello world"
Reversed: DLROW OLLEH
Length: 11
```
Output file `output.txt` should contain:
```
input=hello world
reversed=DLROW OLLEH
length=11
```
